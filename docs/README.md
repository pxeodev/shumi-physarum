# Physarum Generator

An interactive mycelial network simulator that produces emergent fungal transport networks. Thousands of independent agents crawl outward from a central hub, depositing chemical trails that self-organize into organic, branching structures.

## What is Physarum?

Physarum polycephalum (slime mold) is a single-celled organism that builds efficient transport networks to connect food sources. This simulator models that behavior: each agent follows simple rules (sense, turn, deposit), but the collective result is a complex, beautiful network that resembles real biological growth.

## Features

- **Real-time simulation** with up to 8,000 agents
- **Deterministic seeds** for reproducible patterns
- **Full parameter control** over colony behavior, chemistry, and rendering
- **8 color presets** plus custom color pickers
- **Multiple export formats**: PNG, WebM video, animated GIF
- **Seed preview system** with bookmarking for batch recording
- **Fullscreen 1080p recording** for landing page backgrounds
- **Cursor interaction** mode for live exploration
- **Stencil mode** for masked compositions with custom artwork

## Quick Start

1. Open `index.html` in any modern browser (Chrome recommended)
2. The simulation starts automatically with seed 12345
3. Use the sidebar controls to adjust parameters
4. Click **Random** to discover new patterns
5. Hit **Save PNG** or **Record WebM** to export

No build step, no dependencies, no installation required.
